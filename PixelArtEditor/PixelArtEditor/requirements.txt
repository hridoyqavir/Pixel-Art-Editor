Compiler:
GCC (MinGW) or any C compiler that supports SDL2 and SDL2_ttf.

Required Libraries:
SDL2 – Simple DirectMedia Layer (for window management, rendering, input handling)

SDL2_ttf – SDL extension library for rendering TrueType fonts

Standard C Libraries Used:
stdio.h – Standard input/output functions

stdlib.h – For dynamic memory, file operations, etc.

stdbool.h – For using bool data types

string.h – For memory and string manipulation

Operating System:
Windows OS recommended (code uses path to Windows system font: C:/Windows/Fonts/arial.ttf)
